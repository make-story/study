# Glob Patterns, 흔히 줄여서 Globs라고 불리는 패턴 매칭  
Globs은 오래전부터 리눅스 운영체제에서 한 번에 여러 개의 파일을 찾을 때 사용해온 패턴 매칭 기법  

Globs에서 가장 많이 사용되는 와일드카드는 * 기호
