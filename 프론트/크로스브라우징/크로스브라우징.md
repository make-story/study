# 사파리(Safari) Date 에러 (Invalid Date)
```javascript
// 아래와 같이 String 포맷을 Date에 넣었을 경우 에러 발생   
new Date('2021-11-15T01:00:00+0900');
new Date('2022-03-25T02:00:59.999+0900');

// 아래와 같이 해줘야 한다.  
new Date('2021-11-15T01:00:00+09:00');

```
