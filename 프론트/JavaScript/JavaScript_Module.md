# JavaScript modules

https://ko.javascript.info/modules-intro

- AMD
  가장 오래된 모듈 시스템 중 하나로 require.js라는 라이브러리를 통해 처음 개발되었습니다.
- CommonJS
  Node.js 서버를 위해 만들어진 모듈 시스템입니다.
- UMD
  AMD와 CommonJS와 같은 다양한 모듈 시스템을 함께 사용하기 위해 만들어졌습니다.
