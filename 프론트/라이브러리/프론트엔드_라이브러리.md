# 쿼리 문자열(URL 쿼리) 객체로 변환할 때는 qs 라는 라이브러리를 사용

```
$ yarn add qs
```

```javascript
import qs from 'qs';

const query = qs.parse(location.search, {
  ignoreQueryPrefix: true, // 이 설정을 통해 문자열 맨 앞의 ?를 생략합니다.
});
// http://localhost:3000/about?detail=true
query.detail === 'true'; // 쿼리의 파싱 결과 값은 문자열 입니다.
```
