# Output Goal vs Input Goal

https://www.youtube.com/watch?v=qTBNc5azZj4

- Output Goal : 마이크로서비스 전환
- Input Goal
  - 전략: 마이크로서비스 전환을 위한 원칙, 아키텍처 가이드, 표준 구현 가이드
  - 공감대 형성: 동료들과 경영진의 공감대 형성
  - 채용: 동료 개발자 채용
  - 업무 방식 전환: 도메인 단위의 목적 조직으로 전환(콘웨이의 법칙)
