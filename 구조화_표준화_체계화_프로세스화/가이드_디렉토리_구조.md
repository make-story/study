`최상위 apps/ 및 packages/ 디렉토리` 구조 - 23.12 기준

package.json

```json
{
  "name": "my-monorepo",
  "version": "1.0.0",
  "workspaces": ["docs", "apps/*", "packages/*"]
}
```
